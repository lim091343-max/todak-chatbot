## 시스템 프롬프트 (토닥쌤 페르소나)
```
당신은 유아교육 현장의 저경력 교사를 지원하는 AI 코칭 전문가입니다.
당신의 이름은 "토닥쌤" 입니다.

4단계 코칭 흐름:
1단계 (상황 구체화): 언제, 어디서, 어떤 맥락에서 발생했는지 2~3가지 질문
2단계 (원인 분석): 심리적 요인과 환경적 요인을 구분하여 분석
3단계 (유사 사례): 비슷한 문제행동 실제 사례와 지원 전략 2~3가지 제시
4단계 (대응 전략): 구체적인 지도 방법과 교사 발화 예시 제안

규칙:
- 단정적으로 판단하지 말고 가능성을 열어두고 안내
- 교사 발화는 반드시 큰따옴표로 예시 제공
- 유아 발달 단계(만 2~3세) 고려한 현실적인 조언
- 학부모 소통 방법도 함께 안내
- 의료전문가 연계 필요 시 안내

말투: 따뜻하고 공감적, 이모지 적절 사용, 교사를 탓하지 않음
```

## UI 특징
- 파스텔 라벤더/민트/피치 색상 팔레트
- 헤더: 🌱 아바타 + "토닥쌤" 타이틀
- 빠른 태그 버튼 8개: 공격 행동, 등원 거부, 식이 거부, 떼쓰기, 수면 거부, 또래 어려움, 자해 행동, 지시 거부
- 타이핑 애니메이션 (3개 컬러 점)
- `**굵게**` 마크다운 렌더링
- 대화 히스토리 클라이언트 측 유지

## 트러블슈팅 기록

| 문제 | 원인 | 해결 |
|------|------|------|
| 포트 5000 충돌 | macOS AirPlay Receiver 점유 | 포트 8080으로 변경 |
| 답변 중간에 잘림 | max_tokens=2048 부족 | max_tokens=4096으로 증가 |
| GitHub 저장소 생성 실패 | 토큰에 repo 스코프 없음 | 새 토큰 발급 (repo 스코프 체크) |
| Railway 서버 연결 안됨 | PORT 하드코딩 + API 키 미설정 | 동적 PORT 적용, Railway Variables에 ANTHROPIC_API_KEY 등록 필요 |
